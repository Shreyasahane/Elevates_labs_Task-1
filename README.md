# Task 1 – User Input and Manual Date Program

## Objective
Create a Python program that:
- Takes user input for name and internship role
- Displays today’s date manually (without using built-in date functions)
- Uses variables instead of hardcoding inside print statements
- Includes comments for better readability

## Files Included
- `hello_world.py` – Python program with user input and manual date
- `screenshot.png` – Output screenshot 

## How to Run

1. Open terminal in the project folder  
2. Run the command:
python hello_world.py
